Restaurant Manager Manual Users

1. Create a new GitHub repository.
2. Upload index.html to the repository root.
3. In Supabase SQL Editor, run supabase-manual-users-setup.sql.
4. In GitHub Settings > Pages:
   Source: Deploy from branch
   Branch: main
   Folder: /root
5. Open the GitHub Pages URL.

Default login:
Username: admin
Password: admin123

Important:
After first login, go to Settings and change the admin password.
